# balatro-legendary-roller

A fully automated Python script that repeatedly restarts a **Balatro** run until a **Legendary Joker** appears at the start of the game.

The script uses image recognition to detect UI elements and game states, then performs automated mouse movements and button presses to navigate the game without user interaction.
Once the target condition is met, the script stops automatically.

## Features

- 🔄 Automatically restarts Balatro runs
- 🖱️ Creates mouse movements and clicks
- 🖼️ Uses PNG template matching to detect game states
- 🧠 Determines the current screen based on image recognition
- ⏹️ Stops automatically when a Legendary Joker is found
- 🤖 Fully hands-free operation
